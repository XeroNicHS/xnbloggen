# Changelog

> **Language**: [🇰🇷 한국어](CHANGELOG.md) | [🇺🇸 English](CHANGELOG.en.md)

## [0.1.0] - 2026-02-04

- 마크다운 기반 정적 블로그 생성기
- `create` 명령어: 새 블로그 프로젝트 초기화
- `new` 명령어: 포스트/페이지 생성
- `build` 명령어: 정적 사이트 빌드
- `server` 명령어: 로컬 개발 서버
- 포스트와 페이지 구분
- 초안(draft) 기능
- Tags, Categories 등 사용자 정의 Taxonomy
- 아카이브 시스템 (연/월/일)
- 페이지네이션
- 커스텀 템플릿
- 코드 하이라이팅
- 포스트 네비게이션 (이전/다음)
- RSS 피드 자동 생성
- Sitemap 자동 생성
